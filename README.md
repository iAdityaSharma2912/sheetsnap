# DEPLOYMENT SEQUENCE

### 1 — Clone

```bash
git clone https://github.com/YOUR_USERNAME/SheetSnap.git
cd SheetSnap
````

### 2 — Install

```bash
npm install
```

### 3 — Configure Environment

Create `.env`:

```env
VITE_OPENAI_API_KEY=your_key_here
```

### 4 — Run

```bash
npm run dev
```

### 5 — Deploy

```bash
vercel
```
